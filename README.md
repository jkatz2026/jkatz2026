  ### Jonah Katz

  **I build reliable, evaluated, secure AI systems** — the layers that make an
  LLM agent trustworthy enough to put in front of a real user: grounding,
  verification, security, and evaluation. Atlanta-based, focused on AI
  implementation and forward-deployed engineering — taking agents from demo to
  production.

  **Selected work**

  - **[reliable-agent](https://github.com/jkatz2026/reliable-agent)** — capstone: a grounded
    question-answering agent that is secured, orchestrated with a
    plan→retrieve→generate→verify loop, guarded with cost/timeout/fallback, and
    evaluated end to end. Runs fully offline.
  - **[injection-defense-gateway](https://github.com/jkatz2026/injection-defense-gateway)** —
    a security gateway that wraps any agent and enforces a declarative policy, so
    a hijacked prompt cannot leak secrets or exfiltrate data regardless of what
    the model decides to do.
  - **[rag-with-citations](https://github.com/jkatz2026/rag-with-citations)** — grounded RAG
    where every answer sentence carries a citation, plus a faithfulness checker
    that flags any claim the sources do not support. Built for the legal and
    finance bar, where a fabricated citation is a liability.
  - **[agent-eval-harness](https://github.com/jkatz2026/agent-eval-harness)** — a
    dependency-free harness that scores agents with an adversarial judge panel,
    because the rare skill is proving an agent works, not just building it.
